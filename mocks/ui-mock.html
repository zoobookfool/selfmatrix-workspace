合意リスト (ui-design-notes v1.3) — ★は 2026-07-06 の変更

  1. 1. 配信特化・カメラ非対応。視聴はオプトイン = 見ていない配信は購読しない
  2. 2.★ 表示モードは注視⇔グリッドの 2 モード切替。グリッドは正方形に詰める配置で、視聴をオプトインした配信だけがタイルになる (未視聴は上部の「視聴できる配信」導線に出るだけで、画面は分割されない)。グリッド中の「強調選択」ON で任意のタイルを選ぶと、選択タイルが大きく並び非選択はミニタイルへ。OFF に戻せば均等グリッドに戻る
  3. 3. 強調選択で 1 枚だけ選ぶと注視モードへ
  4. 4. ミニタイルは実映像を流し続ける (simulcast 低解像度層)
  5. 5. ミニタイル位置は上下左右から選択
  6. 6. 話者表示は StreamKit 風オーバーレイ (緑リング/ミュートアイコン、ピルはドラッグ可 — 掴んで動かせます)
  7. 7.★ ポップアウトの主対象は通話 UI 全体 (目的: 通話 UI とチャット UI を同時に見て・触る)。モックではフローティング疑似窓で表現 — 実物は別 OS ウィンドウ。配信単体のポップアウト (実装済み・再接続なし) は補助機能として残す
  8. 8. シェル: サーバー枠・チャンネル枠とも上下左右 (上下配置時はチップ列)
  9. 9. 既定は Discord 風: 枠は左、ミニタイル下、注視モード
  10. 10.★ 配置セットアップの案内は SelfMatrix の初回利用時に 1 回だけ (デモボタンで表示)。通話参加時には毎回出さない
  11. +. ミニタイルに名前ピルは出さない / 画面共有で自動レイアウト切替はせずトースト通知
